# 26-CPP — CERN STEAM Academy 2026, C++ course

Material for the C++ course at the CERN STEAM Academy 2026 (Matthias Kretz).

## Contents

- [`exercise-example/`](exercise-example/) — example exercise (n-body simulation + a `std::sin` lookup-table benchmark). It builds out of the box on the course laptops with GCC 16 (C++26), Google Benchmark and SFML 3.1, both fetched automatically via CMake `FetchContent`.

## Course laptops

The environment (laptops, compiler modules, Google Benchmark, real-time scheduling and `perf` setup) is documented at <https://stac.docs.cern.ch/>.
